# Question 1 – Functions (Compulsory) (15 Marks)

Create a function named `calculate_grade(marks)` that accepts a student's marks and returns the grade using the following criteria:

| Marks        | Grade |
| ------------ | ----- |
| 80 and above | A     |
| 60 – 79      | B     |
| 40 – 59      | C     |
| Below 40     | D     |

Call the function and display the grade.

# Question 3 – Dictionaries (15 Marks)

Create a dictionary containing the marks of a student in three subjects.

Example:

Write a program to:

- Calculate the total marks.

# 🧩 Assignment 10: Working with Files in Python

## 📘 Instructions

1. Write each program in a separate Python file.
2. Create the required text files in the same folder as your Python program.
3. Display clear output messages.
4. Close the file after use (or use the `with` statement).
5. Test each program before submission.

---

## 📝 1️⃣ Create and Write to a File

**Problem:**

Write a Python program to create a file named `student.txt` and write the following text into it:

```text
Name: Ahmed
Class: 10
Grade: A
```

Display a message after writing the data successfully.

### Example Output

```text
Data written successfully.
```

---

## 📖 2️⃣ Read and Display File Contents

**Problem:**

Write a Python program to read and display all contents of the file `student.txt`.

### Example Output

```text
Name: Ahmed
Class: 10
Grade: A
```

---

## ➕ 3️⃣ Append Data to a File

**Problem:**

Write a Python program to add the following line to `student.txt` without deleting existing content:

```text
School: ABC Public School
```

Display the updated contents of the file.

### Example Output

```text
Name: Ahmed
Class: 10
Grade: A
School: ABC Public School
```

---

## 🔢 4️⃣ Count the Number of Lines in a File

**Problem:**

Write a Python program to count and display the total number of lines present in `student.txt`.

### Example Output

```text
Total Lines = 4
```

---

## 🔍 5️⃣ Count the Number of Words in a File

**Problem:**

Write a Python program to count the total number of words present in `student.txt`.

### Example Output

```text
Total Words = 10
```

_(The actual count may vary depending on the file content.)_

---

## 📋 6️⃣ Copy Contents from One File to Another

**Problem:**

Write a Python program to copy all contents from `student.txt` to a new file named `backup.txt`.

Display a success message after copying.

### Example Output

```text
File copied successfully.
```

---

## 🔎 7️⃣ Search for a Word in a File

**Problem:**

Write a Python program to check whether the word `"Ahmed"` exists in `student.txt`.

### Example Output

```text
Word found in the file.
```

or

```text
Word not found in the file.
```

---

## 🗂️ 8️⃣ Store Multiple Student Names in a File

**Problem:**

Write a Python program to store the following names in a file named `students.txt`:

```text
Ahmed
Fatima
Rahul
Aisha
Zeeshan
```

Then read and display all names from the file.

### Example Output

```text
Ahmed
Fatima
Rahul
Aisha
Zeeshan
```

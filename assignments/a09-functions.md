# 🧩 Assignment 09: Working with Functions in Python

## 📘 Instructions

1. Write each program in a separate Python file.
2. Use meaningful function names.
3. Call the function after defining it.
4. Display clear output messages.
5. Use parameters and return values wherever required.

---

## 🔹 1️⃣ Create Your First Function

**Problem:**

Create a function named `welcome()` that displays the message:

```text
Welcome to Python Functions
```

Call the function to display the message.

### Example Output

```text
Welcome to Python Functions
```

## ➕ 2️⃣ Function to Add Two Numbers

**Problem:**

Create a function `add_numbers(a, b)` that takes two numbers as parameters and displays their sum.

### Example Output

```text
First Number = 10
Second Number = 20

Sum = 30
```

## ✖️ 3️⃣ Function to Calculate Area of a Rectangle

**Problem:**

Create a function `area(length, width)` that calculates and displays the area of a rectangle.

### Formula

```text
Area = Length × Width
```

### Example Output

```text
Length = 10
Width = 5

Area = 50
```

## ⭐ Bonus Question

### Student Grade Calculator

**Problem:**

Create a function `grade(marks)` that returns the grade according to the following rules:

| Marks        | Grade |
| ------------ | ----- |
| 80 and above | A     |
| 60 to 79     | B     |
| 40 to 59     | C     |
| Below 40     | D     |

Display the returned grade.

### Example Output

```text
Marks = 72

Grade = B
```
